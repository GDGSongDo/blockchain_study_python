This is CHAIN HELPER

Possible URLs for command - localhost:5000/

  1. /chains - to show whole chains
  2. /transactions - to show TXs list before new block
  3. /transactions/new - to make a new TX
  4. /transactions/all - to show whole TXs from the start
  5. /mine - to mine a new block (writing ledger automatically)
  6. /write - to write chains to chains.json
